- i3 (as well as betterlockscreen)
- dunst
- polybar
- pamixer + pavucontrol
- brave
- firefox
- keepassxc
- valent (modern KDEConnect)
- zsh (ohmyzsh)
- lyrebird
- LyX
- calibre
- flameshot
- solaar
